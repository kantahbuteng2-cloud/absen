# Absensi Apel System

Sistem absensi apel digital untuk Lapas Kelas IIB Baubau.

## Fitur
- Scan QR Code untuk absensi
- Pendaftaran perangkat
- Validasi lokasi
- PWA support
- Mobile-first design

## Setup
1. Deploy Google Apps Script dengan kode dari `gas-code.js`
2. Ganti `YOUR_DEPLOYMENT_ID` di HTML dengan URL deployment Anda
3. Upload ke GitHub Pages

## Deployment
- HTML: GitHub Pages
- Backend: Google Apps Script
- Database: Google Spreadsheet
